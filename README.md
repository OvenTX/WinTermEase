# WinTermEase

> 一款给 Windows 用起来很顺手的 **串口 + SSH 双栈终端工具**。  
> 轻一点，快一点，顺眼一点，还带一条很能打的快捷指令栏。

## 这是什么？

**WinTermEase** 是一个基于 **.NET 8 + WPF + WebView2 + xterm.js** 的 Windows 终端应用，专门面向日常开发、调试、板卡联调、设备排障和远程登录场景。

它把你常用的几件事放进了一个窗口里：

- **串口连接**：连开发板、模组、工控设备都很顺手
- **SSH 连接**：支持密码和私钥登录
- **多标签页**：多个会话并排开工，切换不抓狂
- **快捷指令栏**：常用命令一键发送，还能按分组管理
- **亮/暗主题切换**：白天清爽，晚上护眼
- **日志保存**：当前会话内容随手落盘

## 亮点功能

| 功能 | 说明 |
| --- | --- |
| 串口终端 | 支持端口、波特率、数据位、校验位、停止位、流控配置 |
| SSH 终端 | 支持密码登录与私钥文件登录 |
| xterm.js 渲染 | 终端体验更像你熟悉的现代 Terminal |
| 历史连接 | 自动记住最近连接，下次打开直接复用 |
| 快捷指令分组 | 把常用命令按场景分类，发送更快 |
| 溢出折叠 | 指令太多时自动收进 `...` 菜单，不挤界面 |
| 状态栏统计 | 实时显示连接状态、RX/TX 流量 |
| 串口关键词高亮 | `error` / `warning` 这类信息会自动着色，更容易盯异常 |
| 右键粘贴保护 | 多行粘贴前会弹确认，减少手滑事故 |
| 自动复制选区 | 选中文本即可复制，终端操作更丝滑 |

## 界面速览

- 顶部工具栏：新建串口 / SSH、保存日志、切换主题、显示或隐藏快捷栏
- 中间标签区：每个连接一个标签页，串口和 SSH 都能一起开
- 主终端区：WebView2 承载 xterm.js
- 底部快捷栏：按分组展示快捷命令，支持新增、编辑、删除、发送
- 最下方状态栏：显示当前连接状态和收发字节数

## 运行环境

- Windows 10 / 11
- .NET 8 SDK
- **Microsoft Edge WebView2 Runtime**

> 如果启动时提示 WebView2 初始化失败，先安装 WebView2 Runtime。

## 快速开始

```powershell
git clone https://github.com/OvenTX/WinTermEase.git
cd WinTermEase
dotnet restore
dotnet build
dotnet run --project WindowsTerminal.csproj
```

## 使用方式

### 1. 新建连接

- 点击 `+ 串口` 新建串口会话
- 点击 `+ SSH` 新建 SSH 会话
- 如果你之前连过，直接从“历史连接”里选一个即可复用

### 2. 管理快捷指令

- 点击快捷栏右侧 `+` 添加快捷命令
- 支持设置：
  - 指令名称
  - 指令内容
  - 分组
  - 发送后是否自动换行
- 右键命令按钮可直接 **发送 / 编辑 / 删除**

### 3. 保存日志

- 选中一个正在使用的标签页
- 点击右上角 `保存日志`
- 会把当前标签页从打开以来接收到的内容保存为 `.log` 或 `.txt`

## 配置文件

应用会把配置保存在：

```text
%AppData%\WindowsTerminal\config.json
```

当前支持持久化的内容包括：

- 历史连接
- 快捷指令
- 主题
- 字体与滚动缓冲相关设置

## 技术栈

- **WPF**：桌面 UI
- **WebView2**：承载终端前端
- **xterm.js**：终端渲染
- **SSH.NET**：SSH 连接
- **System.IO.Ports**：串口通信
